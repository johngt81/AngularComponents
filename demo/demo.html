Directives

show-more-or-less

Truncate long texts and make them collapsable (comes with default values).
Code
<show-more-or-less data-text="12345 abcde 67890"
    data-max-length="16"
</show-more-or-less>
<br />
<show-more-or-less data-text="12345 abcde 67890"
    data-max-length="16"
    data-show-more-text="[more]"
    data-show-less-text="[less]"
    data-truncate-symbol="">
</show-more-or-less>
Output

Filters

firstWord

Returns the first word using space as the default word separator.
Code
{{ "Lorem.ipsum.ad.his.scripta.blandit.partiendo" | firstWord:'.' }}
Output
{{ "Lorem.ipsum.ad.his.scripta.blandit.partiendo" | firstWord:'.' }}
